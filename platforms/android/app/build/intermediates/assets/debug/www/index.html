Willkommen bei Remind Me!

Mit unserer App können Sie ganz einfach Aufgaben erstellen und sich erinnern lassen.

Gehen Sie dabei wie folgt vor:
Wählen Sie zunächst eine der vordefinierten Kategorien oder erstellen Sie bei Bedarf eine neue.
Anschließend schreiben Sie sich ihre Notiz bzw. Aufgabe auf, an die Sie erinnert werden möchten.
Darauffolgend können Sie ein Datum und eine Uhrzeit angeben, zu der Sie per Benachrichtigung erinnert werden möchten.
Bei Bedarf können Sie auch ein Foto aus ihrer Galerie anhängen oder ein Foto aufnehmen.
Zuletzt folgt noch ein Klick auf "Remind Me!" und schon sind Sie fertig!
